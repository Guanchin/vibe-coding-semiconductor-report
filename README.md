## 分析方法
本研究採用 **Z-score 標準化平均法**：
- 先對各股票個別標準化。
- 消除不同股價之間的尺度差異。
- 再將同一類股中的股票進行平均。
- 建立類股綜合走勢圖。  [oai_citation:9‡生成式 AI期中報告_(2).pdf](sediment://file_0000000090687206bd71372e965f1ce9)

### 方法目的
使用 Z-score 的目的，是讓高價股與低價股可以放在相同基準下比較，更容易觀察相對強弱與整體趨勢，而不是只看絕對價格高低。  [oai_citation:10‡生成式 AI期中報告_(2).pdf](sediment://file_0000000090687206bd71372e965f1ce9)
